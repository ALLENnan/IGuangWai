# IGuangWai
#####项目功能简介
i广外，一个类似论坛、贴吧的信息交流平台，包括了学习区，生活区，活动区，新生区等功能板块，具备不限于以下功能，注册登录、用户个人管理，浏览帖子，发布帖子、回复帖子、贴子管理、收藏搜索帖子等...
